# oPanel (Admin Only)

Bu proje yalnizca admin paneli ve API backend olaraq calisir.

## Calistirma

1. Bagimliliklari qurun:
```bash
npm install
```

2. Admin paneli (Vite) calistirin:
```bash
npm run sys
```

3. Backend API calistirin:
```bash
npm run dev:server
```

## Gelistirme

- `npm run dev`: backend + admin panel beraber calisir
- `npm run build`: admin panel build
- `npm run lint`: lint kontrolu

## Notlar

- Dokumantasyon sablonu: `public/example-index.html`
- Sitemap yapisi: `public/sitemap.json`
